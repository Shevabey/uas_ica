### How to run this project
Clone repositorie
```bash
git clone https://github.com/Shevabey/uas_ica.git

```
Change directory
```bash
cd uas_ica

```
Instal dependecies
```bash
npm install

```
Start Project
```bash
npm start

```
Port development
```bash
http://localhost:8080/

```
